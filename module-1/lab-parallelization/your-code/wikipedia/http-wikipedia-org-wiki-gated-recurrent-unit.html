Gated recurrent unit

\n
\n
From Wikipedia, the free encyclopedia
\n
\n
\n
\nJump to navigation\nJump to search\n

Gated recurrent units (GRUs) are a gating mechanism in recurrent neural networks, introduced in 2014 by Kyunghyun Cho et al.[1] The GRU is like a long short-term memory (LSTM) with a forget gate,[2] but has fewer parameters than LSTM, as it lacks an output gate.[3] \nGRU\'s performance on certain tasks of polyphonic music modeling, speech signal modeling and natural language processing was found to be similar to that of LSTM.[4][5] GRUs have been shown to exhibit better performance on certain smaller and less frequent datasets.[6][7]\n

However, as shown by Gail Weiss, Yoav Goldberg and Eran Yahav, the LSTM is "strictly stronger" than the GRU as it can easily perform unbounded counting, while the GRU cannot. That\'s why the GRU fails to learn simple languages that are learnable by the LSTM.[8]\n

Similarly, as shown by Denny Britz, Anna Goldie, Minh-Thang Luong and Quoc Le of Google Brain, LSTM cells consistently outperform GRU cells in "the first large-scale analysis of architecture variations for Neural Machine Translation."[9]\n

Architecture[edit]

\n

There are several variations on the full gated unit, with gating done using the previous hidden state and the bias in various combinations, and a simplified form called minimal gated unit.[10]\n

The operator \n\n\n\n\xe2\x8a\x99\n\n\n{\\displaystyle \\odot }\n\n denotes the Hadamard product in the following.\n

\n

Fully gated unit[edit]

\n
Gated Recurrent Unit, fully gated version
\n

Initially, for \n\n\n\nt\n=\n0\n\n\n{\\displaystyle t=0}\n\n, the output vector is \n\n\n\n\nh\n\n0\n\n\n=\n0\n\n\n{\\displaystyle h_{0}=0}\n\n. \n

\n
\n\n\n\n\n\n\n\n\nz\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nW\n\nz\n\n\n\nx\n\nt\n\n\n+\n\nU\n\nz\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nb\n\nz\n\n\n)\n\n\n\n\n\nr\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nW\n\nr\n\n\n\nx\n\nt\n\n\n+\n\nU\n\nr\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nb\n\nr\n\n\n)\n\n\n\n\n\n\n\n\nh\n^\n\n\n\n\nt\n\n\n\n\n\n=\n\n\xcf\x95\n\nh\n\n\n(\n\nW\n\nh\n\n\n\nx\n\nt\n\n\n+\n\nU\n\nh\n\n\n(\n\nr\n\nt\n\n\n\xe2\x8a\x99\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n)\n+\n\nb\n\nh\n\n\n)\n\n\n\n\n\nh\n\nt\n\n\n\n\n\n=\n(\n1\n\xe2\x88\x92\n\nz\n\nt\n\n\n)\n\xe2\x8a\x99\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nz\n\nt\n\n\n\xe2\x8a\x99\n\n\n\n\nh\n^\n\n\n\n\nt\n\n\n\n\n\n\n\n\n{\\displaystyle {\\begin{aligned}z_{t}&=\\sigma _{g}(W_{z}x_{t}+U_{z}h_{t-1}+b_{z})\\\\r_{t}&=\\sigma _{g}(W_{r}x_{t}+U_{r}h_{t-1}+b_{r})\\\\{\\hat {h}}_{t}&=\\phi _{h}(W_{h}x_{t}+U_{h}(r_{t}\\odot h_{t-1})+b_{h})\\\\h_{t}&=(1-z_{t})\\odot h_{t-1}+z_{t}\\odot {\\hat {h}}_{t}\\end{aligned}}}\n\n

Variables\n

\n
  • \n\n\n\n\nx\n\nt\n\n\n\n\n{\\displaystyle x_{t}}\n\n: input vector
    • \n
  • \n\n\n\n\nh\n\nt\n\n\n\n\n{\\displaystyle h_{t}}\n\n: output vector
    • \n
  • \n\n\n\n\n\n\n\nh\n^\n\n\n\n\nt\n\n\n\n\n{\\displaystyle {\\hat {h}}_{t}}\n\n: candidate activation vector
    • \n
  • \n\n\n\n\nz\n\nt\n\n\n\n\n{\\displaystyle z_{t}}\n\n: update gate vector
    • \n
  • \n\n\n\n\nr\n\nt\n\n\n\n\n{\\displaystyle r_{t}}\n\n: reset gate vector
    • \n
  • \n\n\n\nW\n\n\n{\\displaystyle W}\n\n, \n\n\n\nU\n\n\n{\\displaystyle U}\n\n and \n\n\n\nb\n\n\n{\\displaystyle b}\n\n: parameter matrices and vector
\n

Activation functions\n

\n
  • \n\n\n\n\n\xcf\x83\n\ng\n\n\n\n\n{\\displaystyle \\sigma _{g}}\n\n: The original is a sigmoid function.
    • \n
  • \n\n\n\n\n\xcf\x95\n\nh\n\n\n\n\n{\\displaystyle \\phi _{h}}\n\n: The original is a hyperbolic tangent.
\n

Alternative activation functions are possible, provided that \n\n\n\n\n\xcf\x83\n\ng\n\n\n(\nx\n)\n\xe2\x88\x88\n[\n0\n,\n1\n]\n\n\n{\\displaystyle \\sigma _{g}(x)\\in [0,1]}\n\n.\n

Alternate forms can be created by changing \n\n\n\n\nz\n\nt\n\n\n\n\n{\\displaystyle z_{t}}\n\n and \n\n\n\n\nr\n\nt\n\n\n\n\n{\\displaystyle r_{t}}\n\n[11]\n

\n
  • Type 1, each gate depends only on the previous hidden state and the bias.\n
    \n\n\n\n\n\n\n\n\nz\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nU\n\nz\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nb\n\nz\n\n\n)\n\n\n\n\n\nr\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nU\n\nr\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nb\n\nr\n\n\n)\n\n\n\n\n\n\n{\\displaystyle {\\begin{aligned}z_{t}&=\\sigma _{g}(U_{z}h_{t-1}+b_{z})\\\\r_{t}&=\\sigma _{g}(U_{r}h_{t-1}+b_{r})\\\\\\end{aligned}}}\n\n
    • \n
  • Type 2, each gate depends only on the previous hidden state.\n
    \n\n\n\n\n\n\n\n\nz\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nU\n\nz\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n)\n\n\n\n\n\nr\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nU\n\nr\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n)\n\n\n\n\n\n\n{\\displaystyle {\\begin{aligned}z_{t}&=\\sigma _{g}(U_{z}h_{t-1})\\\\r_{t}&=\\sigma _{g}(U_{r}h_{t-1})\\\\\\end{aligned}}}\n\n
    • \n
  • Type 3, each gate is computed using only the bias.\n
    \n\n\n\n\n\n\n\n\nz\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nb\n\nz\n\n\n)\n\n\n\n\n\nr\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nb\n\nr\n\n\n)\n\n\n\n\n\n\n{\\displaystyle {\\begin{aligned}z_{t}&=\\sigma _{g}(b_{z})\\\\r_{t}&=\\sigma _{g}(b_{r})\\\\\\end{aligned}}}\n\n

Minimal gated unit[edit]

\n

The minimal gated unit is similar to the fully gated unit, except the update and reset gate vector is merged into a forget gate. This also implies that the equation for the output vector must be changed:[12]\n

\n
\n\n\n\n\n\n\n\n\nf\n\nt\n\n\n\n\n\n=\n\n\xcf\x83\n\ng\n\n\n(\n\nW\n\nf\n\n\n\nx\n\nt\n\n\n+\n\nU\n\nf\n\n\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nb\n\nf\n\n\n)\n\n\n\n\n\n\n\n\nh\n^\n\n\n\n\nt\n\n\n\n\n\n=\n\n\xcf\x95\n\nh\n\n\n(\n\nW\n\nh\n\n\n\nx\n\nt\n\n\n+\n\nU\n\nh\n\n\n(\n\nf\n\nt\n\n\n\xe2\x8a\x99\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n)\n+\n\nb\n\nh\n\n\n)\n\n\n\n\n\nh\n\nt\n\n\n\n\n\n=\n(\n1\n\xe2\x88\x92\n\nf\n\nt\n\n\n)\n\xe2\x8a\x99\n\nh\n\nt\n\xe2\x88\x92\n1\n\n\n+\n\nf\n\nt\n\n\n\xe2\x8a\x99\n\n\n\n\nh\n^\n\n\n\n\nt\n\n\n\n\n\n\n\n\n{\\displaystyle {\\begin{aligned}f_{t}&=\\sigma _{g}(W_{f}x_{t}+U_{f}h_{t-1}+b_{f})\\\\{\\hat {h}}_{t}&=\\phi _{h}(W_{h}x_{t}+U_{h}(f_{t}\\odot h_{t-1})+b_{h})\\\\h_{t}&=(1-f_{t})\\odot h_{t-1}+f_{t}\\odot {\\hat {h}}_{t}\\end{aligned}}}\n\n
\n

  • \n\n\n\n\nh\n\nt\n\n\n\n\n{\\displaystyle h_{t}}\n\n: output vector
    • \n
  • \n\n\n\n\n\n\n\nh\n^\n\n\n\n\nt\n\n\n\n\n{\\displaystyle {\\hat {h}}_{t}}\n\n: candidate activation vector

  • \n\n\n\n\nf\n\nt\n\n\n\n\n{\\displaystyle f_{t}}\n\n: forget vector
    • \n
