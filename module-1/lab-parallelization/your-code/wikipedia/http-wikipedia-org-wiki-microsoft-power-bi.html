Power BI is a business analytics service by Microsoft. It aims to provide interactive visualizations and business intelligence capabilities with an interface simple enough for end users to create their own reports and dashboards.[1]\n

General[edit]

\n

Power BI provides cloud-based BI (business intelligence) services, known as "Power BI Services", along with a desktop based interface, called "Power BI Desktop". It offers data warehouse capabilities including data preparation, data discovery and interactive dashboards.[2]\nIn March 2016, Microsoft released an additional service called Power BI Embedded on its Azure cloud platform.[3] One main differentiator of the product is the ability to load custom visualizations.\n

\n

History[edit]

\n

This application was originally conceived by Thierry D\'Hers and Amir Netz of the SQL Server Reporting Services Team at Microsoft.[4] It was originally designed by Ron George in the summer of 2010 and named Project Crescent.[5] Project Crescent was initially available for public download on July 11, 2011 bundled with SQL Server Codename Denali.[6] Later renamed to Power BI it was then unveiled by Microsoft in September 2013 as Power BI for Office 365.[7] The first release of Power BI was based on the Microsoft Excel\xe2\x80\x93based add-ins: Power Query, Power Pivot and Power View. With time, Microsoft also added many additional features like Question and Answers, enterprise level data connectivity and security options via Power BI Gateways.[2] Power BI was first released to the general public on July 24, 2015.[8]\n

In February 2019, Gartner confirmed Microsoft as Leader in the "2019 Gartner Magic Quadrant for Analytics and Business Intelligence Platform" as a result of the capabilities of Power BI platform.[9] This represented the 12th consecutive year of recognition of Microsoft as Leading vendor in this Magic Quadrant category (beginning 3 years before this tool was even created).[10]\n

\n

Key components[edit]

\n

Key components of the Power BI ecosystem comprises: \n

\n
Power BI Desktop
\n
The Windows-desktop-based application for PCs and desktops, primarily for designing and publishing reports to the Service.
\n
Power BI Service
\n
The SaaS-based (software as a service) online service. This was formerly known as Power BI for Office 365, now referred to as PowerBI.com, or simply Power BI.
\n
Power BI Mobile Apps
\n
The Power BI Mobile apps for Android and iOS devices, as well as for Windows phones and tablets.
\n
Power BI Gateway
\n
Gateways used to sync external data in and out of Power BI and are required for automated refreshes. In Enterprise mode, can also be used by Flows and PowerApps in Office 365.
\n
Power BI Embedded
\n
Power BI REST API can be used to build dashboards and reports into the custom applications that serves Power BI users, as well as non-Power BI users.
\n
Power BI Report Server
\n
An On-Premises Power BI Reporting solution for companies that won\'t or can\'t store data in the cloud-based Power BI Service.
\n
Power BI Premium
\n
Capacity-based offering that includes flexibility to publish reports broadly across an enterprise, without requiring recipients to be licensed individually per user. Greater scale and performance than shared capacity in the Power BI service
\n
Power BI Visuals Marketplace
\n
A marketplace of custom visuals and R-powered visuals.[11]
\n

Example[edit]

\n

Transport for London use Power BI to show when their stations are busiest.[12]\n
