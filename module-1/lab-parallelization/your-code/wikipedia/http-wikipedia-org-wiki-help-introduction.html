\nIntroduction to Wikipedia
\n
\n

Welcome to Wikipedia! Anyone can edit almost every page, and millions already have.\n

\nThis page takes you through a set of tutorials aimed at complete newcomers interested in contributing. It covers all the basics, and each tutorial takes only a few minutes, so you could become a proficient Wikipedian in no time!
\n
\n

\nThe Wiki markup source editor shows the underlying page source code, and works like a plain text file. Links and other items are indicated using simple code like this: [[Earth]].
